# 关于其他开源项目

关于该文档的开发必须要感谢众多的开源项目。以下这些开源项目极大的提升了我的开发效率：

  • vue-admin-template，该项目使用的前端模板，基于该开源项目我在很短的时间内就实现了我想要的功能。当然这得益于我之前已经学过相关的技术栈，这包括 vue 及其附带的 vue-cli、vuex 等。
  • vuepress-personal，该项目提供了一个完整的基于 vuepress 的项目结构，让我只需要专注于文档本身（Markdown）的写作即可。
  • pyechart，使用 Python 构建 echart 的 options，让图标的绘制简单了很多，该项目让后端定制前端有了更高的可行性。不过，由于该项目不支持Python3.5（主要是因为typehint），我修改了Python3.5中语法不支持的地方（削弱了类型支持），使其支持了Python3.5，额外发布了 pyechart35。
  • fitlog，fitlog是一款在深度学习训练中用于辅助用户记录日志和管理代码的工具，该工具中使用 git 的行为和对前端的实现给了我很多灵感，让我最终能够很好的完成试验管理部份的逻辑。
