# 介绍

在开始前，我们首先来看一下 thexp 能帮你完成什么。

thexp 首先提供了一套完整的实验框架。这一套实验框架中的大部分功能已经完成了解耦，在单独使用的情况，能够帮助你快速的完成 超参数声明，变量记录，日志输出，模型保存，随机种子管理，以及对 数据集加载形式的灵活控制。点击这些链接可以帮助你快速的了解如何将这些功能单独使用在你已经完成了一半的实验上。

thexp 远不止提供了这些功能。thexp 还提供了一套基于 git 的实验管理方案，通过该方案以及相关的函数 api，你将不必再思考实验过程中的数据存储位置如何组织，试验结果以什么样的形式记录——thexp中通过约定和接口帮助你完成这一切。同时，基于 git，thexp 赋予了复现你任意试验代码的可能，你可以打包或者重置你任意时间的代码快照。

上述的这些功能实际上可以用于任何类型的实验中，而针对pytorch中的深度学习试验，thexp 还额外提供了一个模块集成了对深度学习的实验管理，使得你只需要关注试验的核心代码，而其余部份，模块通过对训练逻辑的抽象，以及内置的比Keras中更细粒度的回调机制帮助你完善的解决。
