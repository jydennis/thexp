# 超参数声明：Params

该模块位于 thexp.frame.params 。Params用于记录各种超参数和配置变量，支持Json、命令行参数等多种方法定义参数。

TIP

该节介绍的功能的全部示例位于 examples/8.params.py 中

# 基本使用

可以通过直接创建该类的实例来创建你需要的超参数。

from thexp import Params

params = Params()
print(params)

# 继承

你也可以通过继承的方式来实现该类（注意：必须要调用父类的初始化方法）：

from thexp import Params
class MyParams(Params):
    def __init__(self):
        super().__init__() # 注意：必须要调用父类的初始化方法
        self.batch_size = 50
        self.topk = (1,2,3,4)
        self.optim = dict(
            lr=0.009,
            moment=0.9
        )

params = MyParams()
print(params)

# 命令行参数

通过Google提供的 Fire库，Params可以很方便的接受命令行参数：

如新建一个文件命名为ap.py



 

from thexp import Params
params = Params()
params.from_args()

WARNING

在试验过程中存在命令行参数敲错的情形，这可能不会影响试验的正常进行（比如已经定义了默认参数，该命令行参数只是额外新增了一个变量），但会导致试验使用的超参数错误，如果你的单次试验周期很长，出现这种问题会很令人沮丧。因此，当通过命令行参数方式定义超参数时候，会对新设置的参数抛出警告用于提醒。

也因此，强烈建议在试验前定义好所有的超参数，这有利于命令行参数时的错误识别和一些其他因素。

此外，所有在Params中的字典会被自动转换为attr类，使得你可以通过引用的方式来获取值（如params.optim.lr）而不是只能通过字典的方式来获取（即params.optim["lr"]）

# 序列化

Params 支持以Json格式序列化超参数：

params.to_json('params.json')
params.from_json('params.json')
1
2

:: warning 保存的数据中只会包括基本的 str/Number等基本类型以及list/dict等数据结构，其他的内容不会被保存，这包括 numpy 或 torch 中的向量（或标量）。因此，请尽可能的只使用基本的、内置的数据类型来定义超参数。 ::

# 限制域

有一些参数可能有显式的声明值域的需求，因此Params中提供了两个方法choice和arange来帮助完成值域的声明：

from thexp import Params
params = Params()
params.choice("dataset","cifar10","cifar100","svhn")
params.arange("thresh",5,0,20)
1
2
3
4

此时，如果为相应的值分配了不在值域范围内的属性，Params会抛出异常。

TIP

此举主要用来更快的报出错误，避免因在试验过程中发现这种低级错误而浪费时间

# 关联绑定

比如，将数据集和模型结构及其对应的部份超参数进行关联，当数据变动到相应值时，会触发相应的连锁反应

params.bind('dataset','mnist','arch','simplenet')
params.bind('dataset','cifar10','arch','cnn13')
params.bind('arch','simplenet','arch_param',dict(feature=128))
params.bind('arch','cnn13','arch_param',dict(feature=256))

params.dataset = 'cifar10'

print(params.arch)
print(params.arch_param)
params.dataset = 'mnist'
print(params.arch)
print(params.arch_param)

# Grid Search

Params中添加了Grid Search方法，可以很简单的生成：

for g in params.grid_search("optim.lr",[0.1,0.01,0.001]):
    for g in g.grid_search("dataset",['cifar10','cifar100','svhn']):
        print(g.dataset,g.thresh)
