# 变量记录：Meter

该模块位于 thexp.frame.meter。Meter 用于在尽可能的不干扰训练逻辑的同时记录变量（并用于日志输出）。

TIP

该节介绍的功能的全部示例位于 examples/2.meters.py 中

# 基本使用

from thexp import Meter
import torch

m = Meter()
m.a = 1
m.b = "2"
m.c = torch.rand(1)[0]

m.c1 = torch.rand(1)
m.c2 = torch.rand(2)
m.c3 = torch.rand(4, 4) # 会只保留第一行
print(m)

# 输出格式控制

Meter可以很方便的控制输出格式，目前基于需求，主要添加了百分比、浮点精度和整数这三种格式化方法，如下：

from thexp import Meter
m = Meter()
m.a = 0.236
m.b = 3.236
m.c = 0.23612312
m.percent(m.a_)
m.int('b')
m.float(m.c_,2)
print(m)

m.a_ 与 'a'

在 meter 中，变量名+下划线实际等价于去掉了下划线的变量名对应的字符串，也即 m.a_ == "a"，在上面的代码中，这两种形式等价，这种实现方式主要用于IDE的自动补全，在变量后添加一个下划线即可以表示该变量名自身。

# 平均值

在很多时候都需要对输出值取平均，使得观察到的结果比较稳定。这由 thexp.frame.meter.AvgMeter 实现。

WARNING

目前考虑到需求，平均值只计算标量浮点类型的值，如果整数型值需要计算平均，请将其转换为浮点数。

from thexp import Meter,AvgMeter

am = AvgMeter()
for j in range(5):
    for i in range(100):
        m = Meter()
        m.percent(m.c_)
        m.a = 1
        m.b = "2"
        m.c = torch.rand(1)[0]

        m.c1 = torch.rand(1)
        m.c2 = torch.rand(2)
        m.c3 = torch.rand(4, 4)
        m.d = [4]
        m.e = {5: "6"}
        # print(m)
        am.update(m)
    print(am)
